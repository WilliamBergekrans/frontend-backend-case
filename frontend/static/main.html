About the Data
  • The size of the circle represents the number of employees for each entity, hover the point to see number of employees.
  • The name of the manager can be seen by hovering over the relevant point.
  • Index is the overall result, between 0 and 100. Entities with null values are left out.
  • Index change is the change in result since the last measurement was made.
  • Response rate is the answer frequency of the entity.
